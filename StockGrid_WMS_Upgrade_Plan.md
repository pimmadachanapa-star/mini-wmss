# StockGrid WMS Upgrade

## Features

-   Admin Login (JWT)
-   User Roles: Admin, Manager, Warehouse, Viewer
-   Real-time Chat (Socket.IO)
-   Notifications
-   Ticket System
-   Activity Log
-   Backup/Restore
-   Supplier & Customer Management
-   Purchase Orders
-   Stock Count / Adjustment
-   Barcode/QR
-   Lot & Expiry
-   Reports (Excel/PDF)

## Recommended Structure

    frontend/
      index.html
      admin/
    backend/
      server.js
      routes/
      socket.js
    database/

## Suggested Stack

-   Frontend: HTML/CSS/JavaScript
-   Backend: Node.js + Express
-   Database: MySQL
-   Real-time: Socket.IO
-   Auth: JWT
