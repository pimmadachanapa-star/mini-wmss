ตั้งค่าการเชื่อมต่อ EmailJS

กฎการแจ้งเตือนอัตโนมัติ

ระบบจะตรวจสอบทุกครั้งหลังรับสินค้า/ตัดสต็อก และส่งอีเมลแจ้งเตือนเฉพาะครั้งแรกที่สินค้าตกต่ำกว่าจุดสั่งซื้อ (จะไม่ส่งซ้ำจนกว่าสต็อกจะกลับมาปกติก่อน)

รายงานสรุปส่งอีเมล

เนื่องจากเว็บไซต์นี้เป็นไฟล์ static (host บน GitHub Pages) จึงไม่สามารถตั้งเวลาส่งอัตโนมัติทุกวันได้เองโดยไม่มีเซิร์ฟเวอร์ — ใช้ปุ่มด้านล่างเพื่อส่งสรุปด้วยตนเองได้ทันที

ประวัติการแจ้งเตือนที่ส่งแล้ว

เวลาประเภทหัวข้อสถานะ
วิธีสมัคร EmailJS (ฟรี): ไปที่ emailjs.com → สมัครสมาชิก → เพิ่ม Email Service (เชื่อม Gmail/Outlook) → สร้าง Email Template กำหนดตัวแปร {{to_email}}, {{subject}}, {{message}} → คัดลอก Public Key, Service ID, Template ID มาใส่ในฟอร์มด้านบน

ล้างข้อมูลทั้งหมด

ข้อมูลทั้งหมดของ StockGrid ถูกบันทึกไว้ในเบราว์เซอร์เครื่องนี้โดยอัตโนมัติ (localStorage) หากต้องการเริ่มต้นใหม่ทั้งหมด (ลบสินค้า คำสั่งซื้อ ประวัติ และการตั้งค่าทุกอย่าง) กดปุ่มด้านล่าง

StockGrid — Mini-WMS (Phase 0–5: Receiving · Orders · Returns · Reports · Email Alerts) · ข้อมูลทั้งหมดบันทึกอัตโนมัติไว้ในเบราว์เซอร์เครื่องนี้ (localStorage) รีเฟรชหน้าแล้วข้อมูลยังอยู่เหมือนเดิม — ใช้ปุ่ม "ล้างข้อมูลทั้งหมด" ในหน้าแจ้งเตือนหากต้องการเริ่มต้นใหม่
